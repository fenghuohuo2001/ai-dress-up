# 提示词模板库

本文件包含所有生成任务的提示词模板。每次生成前必须读取本文件，禁止自行编造提示词。

---

## 一、换装图片提示词

### 正向提示词（固定模板，原样使用）

```
请根据以下两张参考图片，生成换装效果图。
- 图1（第一张）：原始角色形象
- 图2（第二张）：目标衣物样式
** 严格遵守以下要求 **：
1. 完全保留图1中角色的面部特征（五官、肤色、发型、头饰）
2. 完全保留图1中角色的身体姿态和动作
3. 完全保留图1中的背景环境和光影效果
4. 仅将图1中角色的服装替换为图2中的款式
5. 服装颜色、图案、材质完全忠实于图2
6. 画面中仅出现一个人物，无多余人物或杂物
7. 整体画风统一，无明显风格突变
8. 服饰贴合身体，人体比例协调，结构正确，头部身体大小、比例合适
9. 无肢体扭曲，无穿模，无多手指，无断肢，姿势自然，画质清晰
```

```
背景变化,场景改变,光影突变,角度变化,透视变化,构图改变,多人,重复人物,
多余人物,杂物,文字,水印,模糊,低清,噪点,服装残缺,服装穿模,颜色偏移,
风格突变,身体畸变,手部畸形,比例失调,换脸,面部重绘
```

## 二、视频转场提示词

### 提示词模板（每次根据原始参考图和换装结果图的实际内容填写 {} 部分）

```
视频参数：5秒 | 720P | 电影级柔光氛围感 | {画风描述，如：二次元偏真人写实国风}

时间轴：
- 0-1.5秒：完全还原参考图1人物原始形象，面部五官、肤色、发型、{原始服装描述}100%保留，背景为{背景描述}完全不变
- 1.5-3秒：原始人物从头到脚完整、自然、协调地向画面{左侧/右侧}走出画面并消失，无任何身体部位残留于画框内，{动态细节，如：长发、衣袂随动作自然飘动}，全程无穿模、无肢体扭曲，背景完全保留无任何变化，人物必须100%走出画框外，连发丝、衣角、脚尖均不可残留。
- 3秒：画面仅留完整背景，零人物、零残影、零轮廓线。此帧必须是纯背景画面，与无人时的背景完全一致。
- 3-5秒：换装后人物从画面{同侧/另一侧}返回，服装替换为{换装后服装描述}，入画顺序为脚部先进入画面，随后身体整体走入，精准还原面料肌理、服饰版型，服饰贴合身体，无穿模、无多手指、无结构错误，光影与背景完全融合
- 第 5 秒最终帧：画面严格 100% 还原参考图 2，人物姿态、服饰、光影、背景完全匹配，无任何偏差。

全程约束：单人画面 | 背景固定 | 光影统一 | 手部五指正常 | 无跳变无卡顿
换装在画框外完成，画面内不出现任何换装中间状态
```

### 填写说明
- `{画风描述}`：根据角色风格填写，如"二次元偏真人写实国风""赛博朋克科幻风""现代都市写实风"
- `{原始服装描述}`：描述参考图1中角色穿的服装，如"白色仙裙、头戴月牙发冠"
- `{背景描述}`：描述参考图1中的背景，如"月下山巅仙境""城市霓虹夜景"
- `{左侧/右侧}`：根据画面构图选择人物走出方向
- `{动态细节}`：根据服装特点描述动态效果，如"长发飘动、裙摆飞扬""披风猎猎作响"
- `{同侧/另一侧}`：人物返回方向，推荐"同一侧"保持连贯
- `{换装后服装描述}`：描述参考图2中的目标服装，如"现代黑色皮衣短裙、银色耳环"

### 负向提示词（已硬编码在 video_api.py 中，无需手动传入）

```
画面硬切，画面扭曲变形，镜头突变，跳变，多个人物，人物变形，背景变化，
光影不一致，色彩失真，模糊，闪烁，穿模，转场生硬，人物残留，半身消失，
换装在画面内完成，人物走出后残影仍留在画面，姿势突变，卡顿，身体残缺
```

### 视频核心约束（必须遵守，不可省略）
1. 原始人物必须彻底走出画面消失，画面呈现完全无人状态
2. 换装在画面外完成，画面内不出现任何换装动作
3. 人物返回时已穿新服装，动作连贯自然
4. 全程单人、背景固定、光影统一、无跳变无卡顿

### API 固定参数（禁止修改）
- `prompt_extend`: false（禁止模型自动扩写）
- `resolution`: 720P
- `duration`: 5秒
