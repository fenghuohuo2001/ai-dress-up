# AI 换装创作技能

## 触发词
开始换装 / 进入换装模式 / 生成换装 / 继续换装 / 恢复创作

---

## 一、禁止事项（最高优先级，违反任何一条都是严重错误）

1. **禁止发送 OSS 链接、http(s) 链接、本地文件路径给用户**。只能发送飞书云文档链接
2. **禁止跳过收费确认**。每次调用付费模型前必须先告知模型名+价格，等用户确认
3. **禁止在生成完成前发送"完成"通知**。图片/视频未拿到结果前，不能说"已完成"
4. **禁止纯文字 prompt 生成换装图**。换装图必须传双参考图（角色原图+衣物图）
5. **禁止使用 wan2.6-i2v-flash 模型**。视频只用 wanx2.1-kf2v-plus
6. **禁止自行发挥，自行改写，严禁主观臆断**

## 三、流程启动与初始化

### 3.1 素材确认
启动后先确认用户手中有什么，判断从哪步开始：

| 用户提供的素材 | 开始步骤 |
|---|---|
| 角色原图 + 衣物图 | → 生成换装图 |
| 角色原图 + 衣物图 + 换装结果图 | → 生成视频 |
| 仅角色名（无图） | → 查素材库 `config/material_lib.yaml`，无图则生成原始形象 |

发送素材确认消息：
```

2. 衣物参考图：{有/无}
3. 换装结果图：{有/无}
我将从「{步骤名}」开始。
```

## 六、错误处理（任何错误都必须通知用户，禁止静默失败）

| 错误类型 | 回复用户 |
|---|---|
| API 限流 | 「API 限流，60秒后自动重试（最多2次）」→ 重试失败则「重试失败，请回复"重新生成"」 |
| 余额不足 | 「账号余额不足。图片¥0.5/张，视频¥7.5/次。请充值后回复"重新生成"」 |
| API Key 无效 | 「API Key 无效，请联系管理员」 |
| 视频超时 | 「视频生成超时（10分钟），请回复"重新生成"」 |
| 飞书上传失败 | 「云盘上传失败，文件在 /tmp/{文件名}，可手动上传」 |
| 其他错误 | 「出错：{错误信息}，请回复"重新生成"」 |

### 恢复指令
- **"重新生成"**：重新执行最近一次失败的步骤
- **"继续"**：跳过当前步骤，执行下一步
- **"继续换装" / "恢复创作"**：读取云文档各章节状态（待填充/已完成），判断进度，向用户确认：
  「检测到上次进度：原始形象✓ 换装图✓ 视频待生成。从「生成视频」继续，确认？」
